🛠 How to Run This Dashboard

Follow these steps once. After that, just open the file and sign in every day.

⚠ Step 1 is one-time only. After completing it, you only need Steps 4 & 5 each day.
1
Enable Google Sheets API & Authorise Origin
Go to console.cloud.google.com → your project → APIs & Services → Library → enable Google Sheets API.
Then: Credentials → your OAuth 2.0 Client → Authorised JavaScript origins → add:
http://localhost:8080
Click Save.
2
Add "Daily Summary" and "Leads" Tabs to Your Sheet
In your Google Sheet click + at the bottom twice and create tabs named exactly:
Daily Summary Leads
3
Save the HTML File to Your Desktop
C:\Users\crm1\Desktop\Personal_Shopper_Dashboard.html
4
Open CMD and Start the Local Server
Press Win + R → type cmd → Enter. Then:
cd Desktop python -m http.server 8080
Keep this window open. Install Python from python.org if needed (free).
5
Open in Chrome & Sign In
http://localhost:8080/Personal_Shopper_Dashboard.html
Sign in with your Crepdog Crew Google account → data loads automatically!
